Supermarket Sales Analysis 

Overview
This is a beginner-level data analysis group project created using **Python, SQL, and Power BI**.
The project analyzes supermarket sales data to identify sales trends, customer behavior,
and product performance.

Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL (MySQL)
- Power BI
- Excel / CSV


Dataset
The dataset contains supermarket transaction details including:
- Invoice ID
- City & Branch
- Customer Type & Gender
- Product Line
- Quantity & Total Sales
- Date & Payment Method

Project Structure
data/ → Raw dataset
sql/ → SQL queries
python/ → Python analysis notebook
powerbi/ → Power BI dashboard


Analysis Performed
- Total revenue and transaction count
- Monthly sales trends
- Sales by city and product line
- Customer type and gender analysis
